# Tropical Freeze ANIM codec status

This branch replaces successful-looking heuristic transforms with a strict decoder boundary: only a layout that has been validated against the supplied corpus is allowed to produce Blender keyframes.

## Supplied corpus

The scanner processed 21 exported character packages and 942 ANIM files, including mirrored copies inside model packages.

| Family | Files |
| --- | ---: |
| `normal_clip` (`0x81`) | 329 |
| `packed_clip_82` (`0x82`) | 537 |
| `packed_state_c1` (`0xC1`) | 2 |
| `packed_state_c2` (`0xC2`) | 74 |

Ninety-four files are in packages without a usable exported skeleton. Eight compact files represent four unique clips mirrored into model packages.

## Implemented

- Strict `RFRM/ANIM` envelope validation.
- Family and frame-count-hint extraction.
- LSB-first active-node mask decoding with explicit SKEL node indices and names.
- The 21-byte/two-frame compact `0x81` layout used by the four supplied Urchin shift clips.
- Its 64-bit smallest-three quaternion representation, retained as WXYZ through the pipeline.
- A reference quaternion encoder for round-trip regression tests and future repacking.
- Typed timeline channels: `rotation_quaternion`, `location`, and `scale`.
- A dedicated Blender importer that writes only typed, verified channels into the Armature already present in the opened `.blend` file.
- Removal of stale timelines generated by the previous heuristic decoder.
- Preservation of old probe output under `legacy_track_decode` for research only.

## Deliberately not marked decoded

The larger `0x81`, `0x82`, `0xC1`, and `0xC2` streams are reported as `pending:*`. Their channel tables, cumulative delta widths, timing records, and bind-pose semantics are not yet established well enough to create trustworthy transforms.

The old implementation guessed descriptor meanings, stream offsets, signed-16 layouts, track order, and Euler rotations. Those guesses are no longer allowed to reach Blender.

## Running the corpus scanner

```bash
python scan_tf_anim_samples.py <folder-containing-character-packages> \
  --output tf_anim_scan_report.json
```

## Tests

```bash
python -m unittest -v test_anim_tf_codec.py
python -m py_compile \
  anim_tf_codec.py \
  anim_tf_codec_patch.py \
  blender_decoded_timeline_patch.py \
  scan_tf_anim_samples.py \
  test_anim_tf_codec.py
```

The regression suite includes a packed key copied from the supplied Urchin sample, in addition to round-trip and node-mask tests.
